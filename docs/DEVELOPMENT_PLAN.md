# Calendex 开发计划

## 📋 项目概述

Calendex 是一个跨平台的桌面日历应用程序，支持与 Radicale 服务器完全同步，具有现代化的用户界面设计和完整的 CalDAV 支持。

## ✅ 当前状态 (v0.1.0)

### 已实现的核心功能

1. **CalDAV 集成**
   - 与 Radicale 服务器的完整同步
   - 符合 RFC 5545 标准的 iCalendar 支持

2. **事件管理**
   - 完整的 CRUD 操作（创建、读取、更新、删除）
   - 事件分类系统
   - 多标签支持
   - 重复事件（每日、每周、每月、每年）
   - 全天事件支持

3. **设计系统**
   - 4 种莫兰迪配色方案
   - 统一的圆角设计系统
   - 完整的暗色模式支持
   - 响应式布局

4. **技术架构**
   - Electron 桌面应用
   - React + Tailwind CSS 前端
   - Express.js 后端 REST API
   - TypeScript 100% 类型覆盖

### 测试结果

- **API 测试**: 9/9 通过 (100%)
- **前端功能**: 所有特性验证通过
- **性能指标**: 
  - 首屏加载 < 2 秒
  - API 响应时间 < 500ms
  - JavaScript 包大小 ~500KB (gzip: ~150KB)

## 🛠️ 开发环境设置

### 前提条件
- Node.js 20+
- pnpm 包管理器

### 开发启动
```bash
# 安装依赖
pnpm install

# 启动后端服务器 (端口 3001)
cd apps/server && pnpm dev

# 启动前端开发服务器 (端口 5173)
cd apps/renderer && pnpm dev
```

### 测试运行
```bash
# 运行自动化测试套件
./run-tests.sh
```
